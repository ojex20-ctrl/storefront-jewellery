SVG bottle placeholders. Generated by hand for the prototype — replace
with real photography when shoots are done. Each file is named after the
product handle (vetiver-studio.svg, saffron-smoke.svg, etc.) and tinted
with the perfume's `flacon` hex from src/lib/products.ts.

The Placeholder component reads the file via background-image (CSS), not
next/image, so an .svg extension works without enabling
`dangerouslyAllowSVG` in next.config.
